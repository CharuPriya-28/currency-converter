# currency-converter
A modern and responsive Currency Converter web application built using HTML, CSS, and JavaScript. It provides real-time currency conversion using a live exchange rate API and supports multiple international currencies through an intuitive user interface

## 📌 Features

- 💰 Convert currencies instantly
- 🌍 Supports 10+ international currencies
- 📈 Real-time exchange rates
- 🎨 Clean and professional baby pink user interface
- 📱 Responsive and user-friendly design
- ⚡ Fast and lightweight
- 🔽 Easy currency selection using dropdown menus
- 🖥️ Runs directly in any modern web browser

## 🚀 How to Run the Project

1. Download or clone this repository.
2. Open the project folder in **Visual Studio Code**.
3. Install the **Live Server** extension (optional but recommended).
4. Open `index.html`.
5. Right-click and select **Open with Live Server**.
6. Enter an amount.
7. Select the source currency.
8. Select the target currency.
9. Click **Convert Currency** to view the result.
